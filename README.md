# BlogSphere

BlogSphere is a vibrant and inclusive blogging platform designed for writers and readers to connect through the art of blogging. It allows users to compose, share, and engage with blog content while exploring various topics and building a supportive community.

## Features

- **Compose and Share Blogs**: Write and publish your thoughts, experiences, and expertise on various topics.
- **Private or Public Posts**: Choose whether to share your content with everyone or keep it within a selected audience.
- **Community Engagement**: Discover and connect with community members by viewing their blogs and activities.
- **Notifications**: Stay updated with notifications for likes, comments, and followers.
- **Explore Blogs Worldwide**: Browse through blogs posted by users from all around the globe.
- **Commenting System**: Engage with posts by leaving comments and joining discussions.

## Technologies Used

This project utilizes a variety of modern technologies to ensure a smooth user experience and robust functionality:

- **Backend**:
  - Node.js
  - Express.js
  - MongoDB
  - Mongoose
  - Passport.js for authentication
  - Body-parser
  - Dotenv for environment variable management

- **Frontend**:
  - EJS (Embedded JavaScript templating)
  - HTML/CSS for styling

## Getting Started

To get a copy of BlogSphere up and running on your local machine, follow these steps:

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/DhirajsGithub/Blog-Share-Store-Website
   ```
2. **Install Dependencies**:
   ```bash
    npm install
    ```
3. **Run the Application**:
    ```bash
    node index.js
    ```
4. **Access the Application**:
 Open your browser and navigate to `http://localhost:5500/` to access BlogSphere.
